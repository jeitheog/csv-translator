📖 Guía paso a paso — cómo crear la app en Shopify Partners

① Crear la app

  1. Ve a partners.shopify.com y crea una cuenta gratuita (si no tienes)
  2. En el dashboard: Apps → Create app → Create app manually
  3. Ponle cualquier nombre (ej: "CSV Traductor") y haz clic en Create

② Configurar las URLs (paso crítico)

Ve a App setup → sección URLs y rellena los dos campos:

App URL

https://csv-translator-gray.vercel.app

Allowed redirection URL(s)

https://csv-translator-gray.vercel.app/oauth_callback.html

⚠️ Si ves http://localhost en alguno de estos campos, sustitúyelo por las URLs de arriba. De lo contrario, Shopify rechazará la conexión.

③ Copiar las credenciales

  1. Haz clic en Save
  2. En la misma página verás el Client ID y el Client secret
  3. Cópialos y pégalos en los campos de arriba → Guardar credenciales

o introduce el token manualmente

¿Cómo obtener el Access Token manualmente?
  1. Ve a tu Shopify Admin → Settings → Apps → Develop apps
  2. Crea una nueva app (dale cualquier nombre)
  3. En Configuration → Admin API scopes: activa write_products y read_products
  4. Haz clic en Install app
  5. Copia el Admin API access token (empieza por shpat_)
